E-Chair

E-Chair presenta una mayor facilidad que las silla de ruedas convencionales.

Contaras
Selector de destino fisico
Selector de destino digital

Arduino Mega 2560

E-Chair cuenta con un Arduino Mega 2560 como microprocesador central
Con su gran cantidad de pines podemos permitirnos utilizar todos los sensores para poder asegurar tanto la seguridad de nuestros clientes como de la precision de movimiento

Esp32

El esp32 es un microcontrolador que cuenta con un modulo WIFI el cual se utiliza para recibir todas las peticiones del usuario

Otras Tecnologias

Este proyecto cuenta con otras tecnologias para su funcionamiento
Conoce todas estas tecnologias y la forma en la que las implementamos en nuestro repositorio de GitHub

Recorridos

Distancias

Nuestro producto mide las distancias recorridas por un sensor opticos el cual registraran cada uno de los pasos de l silla

Protecciones

Gracias a nuestro producto usted podra sentirce lo mas seguro posible, contara con sensores de proximidad los cuales preveran cualquier accidente para lograr detener y evitar el choque, como tambien de un acelerometro con el cual s podra regresar a la ruta del recorrido en caso de una desviasion
